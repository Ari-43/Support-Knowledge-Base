How to use OptiFine Performance Presets v1.0.0 for MC 1.18.2

If you need help following this guide, direct message Ari RERA 43#0044 on Discord

  1. Download this .zip file (already done if you're viewing this guide)
  2. Extarct it
  3. Open the extracted folder
  4. Copy the folder named .minecraft inside the unzipped folder
  5. Close Minecraft if it isn't already closed
  6. Press + R
  7. Paste %AppData% into the window that opened (Run menu)
  8. Paste the .minecraft folder you copied earlier into the %AppData% folder you just opened. Recopy the .minecraft folder if needed.
  9. Press yes if you are prompted to replace files
  10. Open Minecraft and verify that all options have been set for maximum performance
